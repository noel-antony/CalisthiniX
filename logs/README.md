# Logs Directory

This directory contains application log files for debugging and monitoring purposes.

## Log Files

- `error.log` - Application errors and exceptions
- `access.log` - API request/response logs
- `database.log` - Database query logs
- `auth.log` - Authentication and session logs

## Usage

Logs are automatically generated during application runtime. Review these files to:
- Debug issues and errors
- Monitor API performance
- Track user authentication events
- Analyze database query patterns
